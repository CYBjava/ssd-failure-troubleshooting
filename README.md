<h1>ssd-failure-troubleshooting</h1>

<h2>Description</h2>
Diagnosed and resolved Windows crashes caused by a failing SSD. I utilized Event viewer, cmd tools like sfc, dism, and chkdsk, and device manager to idenitfy and mitigate the root problem
<br />

<h2>Summary</h2>

After Windows failed to format a USB, the PC would crash frequently after removing the USB. After I troubleshot the problem, I concluded that a new SSD was required along with a reinstall of Windows 11
